# Buttons
#### Using Bootstrap v5.0
[Bootstrap Documentation](https://getbootstrap.com/docs/5.0/components/buttons/)


### Default buttons

Use these buttons in most situations. If you don't want the animation, you can leave out the .btn-animated class.
<Canvas>
    <button class="btn btn-default btn-animated"><span>Secondary</span></button>
    <button class="btn btn-primary btn-animated"><span>Primary</span></button>
    <button class="btn btn-link">Tertiary</button>
</Canvas>


### Animated

The animated buttons need a span around the text!

<button type="button" class="btn btn-primary btn-sm btn-animated">
<span>Small button</span>
</button>
<button type="button" class="btn btn-primary btn-rg btn-animated">
<span>Regular button</span>
</button>
<button type="button" class="btn btn-primary btn-md btn-animated">
<span>Default button</span>
</button>
<button type="button" class="btn btn-primary btn-lg btn-animated">
<span>Large button</span>
</button>
