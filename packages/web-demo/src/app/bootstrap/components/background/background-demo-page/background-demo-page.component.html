All these background-color classes can be used with the class "text-auto". This will automatically calculate the foreground color (black or white).
If you omit the text-auto class, you will have to make sure the text is still readable by yourself.
Bootstrap suggests using "text-dark" or "text-white" in their docs. (See link "Bootstrap Doc" above.)
